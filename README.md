# 뉴스레터 웹앱

## 프로젝트 소개
키워드 기반으로 최근 1주일치 한국 신문기사를 수집하여 뉴스레터 형태로 제공하는 웹 애플리케이션입니다.

## 주요 기능
- 키워드 기반 뉴스 검색
- 최근 1주일치 기사 수집
- 뉴스레터 생성 및 이메일 발송
- 사용자 맞춤 설정

## 기술 스택
- Frontend: React.js, TypeScript, Tailwind CSS
- Backend: Python FastAPI
- Database: PostgreSQL
- 기타: Docker, AWS/GCP
